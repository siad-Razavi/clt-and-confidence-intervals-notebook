# clt-and-confidence-intervals-notebook
A Jupyter notebook exploring confidence intervals, Gaussian assumptions, and the Central Limit Theorem through simulation and real-data examples. # CLT and Confidence Intervals Notebook

This repository contains a Jupyter notebook that explores key concepts in statistical inference, with a focus on confidence intervals, Gaussian assumptions, and the Central Limit Theorem (CLT).

The notebook combines theoretical explanations, simulations, and applied examples to illustrate how confidence intervals can be constructed and interpreted in practice.

## Contents

The notebook covers:

- Confidence intervals for the mean under Gaussian assumptions
- Normality assessment using histograms and Q-Q plots
- Confidence intervals based on Student’s t-distribution
- Confidence intervals for variance using the chi-square distribution
- Simulation-based illustration of the Central Limit Theorem
- CLT-based confidence intervals for non-Gaussian data
- Applied examples on real datasets, including trip-duration analysis

## Objectives

The main goals of this notebook are to:

- understand the role of confidence intervals in statistical analysis
- examine when Gaussian assumptions are reasonable
- illustrate how the Central Limit Theorem supports inference in practice
- compare theoretical and empirical behavior through simulation
- apply statistical inference tools to real-world data
